# 矿井不安全行为识别检测调度系统 - 使用说明

## 📖 系统概述

本系统是一个专业的矿井安全监控管理平台，采用现代化的前端技术栈，提供实时监控、告警管理和数据分析三大核心功能。系统具有工业风格的界面设计，完美适配矿井监控室的使用场景。

## 🚀 系统启动

### 方法一：本地开发环境
1. 确保已安装 Node.js 18+ 版本
2. 打开终端，进入项目目录
3. 运行以下命令：
   ```bash
   npm install
   npm run dev
   ```
4. 在浏览器中访问 `http://localhost:3000`

### 方法二：直接访问部署版本
- 如果已部署到 Vercel，直接访问提供的 URL
- 建议使用 Chrome 浏览器以获得最佳体验

## 🎛️ 功能使用指南

### 1. 实时监控模块

#### 设备管理
- **设备列表**：左侧显示所有监控设备，包括摄像头状态和最后在线时间
- **设备状态**：
  - 🟢 绿色：设备在线
  - 🟡 橙色：设备警告
  - 🔴 红色：设备离线
- **设备选择**：点击设备卡片可切换查看不同监控点

#### 摄像头功能
- **开启摄像头**：点击"开启摄像头"按钮调用本地摄像头
- **视频显示**：视频将在中央区域实时显示
- **截图功能**：点击"截图"按钮可保存当前画面
- **关闭摄像头**：使用完毕后点击"关闭摄像头"释放资源

#### 环境监测
- **传感器数据**：右侧实时显示环境参数
- **数据类型**：
  - 瓦斯浓度
  - 粉尘浓度
  - 环境温度
  - 环境湿度
  - 氧气浓度
  - 风速
- **异常提醒**：数据超标时会自动标红并显示警告

### 2. 告警管理模块

#### 告警筛选
- **类型筛选**：可选择"未戴安全帽"、"跨越轨道"、"气体超标"
- **状态筛选**：按"未处理"、"处理中"、"已解决"筛选
- **优先级筛选**：按"高"、"中"、"低"优先级筛选
- **时间范围**：可选择最近24小时、7天、30天
- **重置筛选**：一键清除所有筛选条件

#### 告警处理
- **查看详情**：点击眼睛图标查看告警详细信息
- **标记处理中**：将未处理告警标记为处理中状态
- **标记已解决**：将处理中告警标记为已解决状态
- **告警优先级**：
  - 🔴 高优先级：需要立即处理
  - 🟡 中优先级：需要尽快处理
  - 🔵 低优先级：可以稍后处理

#### 告警详情
- **基本信息**：告警类型、标题、描述、位置、时间
- **设备信息**：相关监控设备的状态信息
- **环境数据**：告警发生时的周边环境参数
- **操作记录**：告警的处理状态和操作历史

### 3. 数据分析模块

#### 关键指标
- **总告警数**：显示本月告警总数
- **已解决告警**：显示处理完成的告警数量
- **解决率**：自动计算告警处理效率
- **在线设备**：显示当前在线的设备数量

#### 趋势分析
- **周告警趋势**：显示最近7天的告警数量变化
- **告警类型分布**：饼图展示不同类型告警的比例
- **设备状态统计**：统计设备在线率和故障率
- **处理效率分析**：显示告警处理的平均时间

#### 环境监测
- **实时数据**：当前各项环境参数的数值
- **历史对比**：与平均值和最大值的对比
- **状态指示**：
  - 🟢 正常：参数在安全范围内
  - 🟡 警告：接近安全阈值
  - 🔴 危险：超过安全阈值

## 🔧 系统配置

### 浏览器要求
- **推荐浏览器**：Chrome 90+、Firefox 88+、Safari 14+
- **功能要求**：需要支持 WebRTC（摄像头调用）
- **分辨率推荐**：1280×720 或更高

### 权限设置
- **摄像头权限**：首次使用时需要授予浏览器摄像头访问权限
- **本地存储**：系统会保存用户的界面偏好设置

## 📞 故障排除

### 常见问题

**Q: 摄像头无法开启？**
A:
1. 检查浏览器是否已授予摄像头权限
2. 确认摄像头未被其他程序占用
3. 尝试刷新页面重新授权

**Q: 页面显示异常？**
A:
1. 清除浏览器缓存
2. 检查网络连接
3. 尝试使用不同的浏览器

**Q: 数据不更新？**
A:
1. 这是演示系统，数据为模拟数据
2. 刷新页面可以重置所有数据
3. 实际部署时需要对接真实数据接口

**Q: 移动端显示问题？**
A:
1. 建议横屏使用以获得更好体验
2. 某些复杂功能在移动端可能受限

## 📊 技术架构

### 前端技术栈
- **框架**：Next.js 16 + TypeScript
- **样式**：Tailwind CSS v4
- **图表**：Recharts
- **图标**：Lucide React
- **状态管理**：React Hooks

- **设备数据**：模拟6个监控摄像头设备

- **传感器数据**：6类环境参数的实时模拟
- **统计数据**：基于模拟数据的图表展示

- **开发环境**：本地 npm run dev
